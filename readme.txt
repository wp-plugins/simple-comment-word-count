A simple plugin to show the number of words in a comment.

== Description ==

This plugin allows you to quickly and simply display the number of words in any of the comments on your WordPress website by simply adding a template tag into you theme.

== Installation ==

To install the plugin simply follow the instructions below:

1. Upload `plugin-name.php` to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place '<?php comment_wordcount(); ?>' in your theme files
